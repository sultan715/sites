Datenschutzerklärung – CamperDB

Informationen zur Verarbeitung personenbezogener Daten bei der Nutzung der App CamperDB.

1. Allgemeine Hinweise

Der Schutz Ihrer persönlichen Daten ist uns ein besonderes Anliegen. Die App „CamperDB“ wurde so entwickelt, dass möglichst wenige personenbezogene Daten verarbeitet werden.

Die Nutzung der App ist grundsätzlich ohne Registrierung und ohne Angabe personenbezogener Daten möglich.

3. Datenverarbeitung in der App

3.1 Lokale Datenspeicherung

Alle von Ihnen eingegebenen Daten werden ausschließlich lokal auf Ihrem Gerät gespeichert. Dazu gehören insbesondere:

  • gespeicherte Campingplätze
  • Reisedaten und Tagebucheinträge
  • Fotos
  • individuelle Einstellungen

Diese Daten verlassen Ihr Gerät nicht und werden nicht an externe Server übertragen.

3.2 Standortdaten

Die App kann Standortdaten verwenden, um Funktionen wie das Speichern von Campingplätzen oder die Navigation zu ermöglichen.

Dabei gilt:

  • Die Nutzung erfolgt ausschließlich nach Ihrer ausdrücklichen Zustimmung.
  • Die Verarbeitung erfolgt ausschließlich auf Ihrem Gerät.
  • Es findet keine Übertragung an Dritte statt.

Sie können die Standortfreigabe jederzeit in den Einstellungen Ihres Geräts widerrufen.

3.3 Zugriff auf Fotos

Die App kann Zugriff auf Ihre Fotos anfordern, um Bilder zu Campingplätzen oder Reisetagen hinzuzufügen.

Dabei gilt:

  • Der Zugriff erfolgt nur nach Ihrer Zustimmung.
  • Es werden nur die von Ihnen ausgewählten Fotos verwendet.
  • Es erfolgt keine automatische Analyse oder Weitergabe.

3.4 Zugriff auf Geräteinformationen

Zur sicheren Speicherung sensibler Daten, zum Beispiel Fahrzeugdaten, nutzt die App den iOS-Schlüsselbund. Diese Daten werden verschlüsselt gespeichert und nicht extern übertragen.

4. Keine Datenübertragung an Dritte

Die App „CamperDB“:

  • überträgt keine Daten an externe Server
  • nutzt keine Cloud-Dienste
  • verwendet keine Tracking-Tools
  • enthält keine Werbung
  • gibt keine Daten an Dritte weiter

5. Externe Dienste

Die App kann zur Navigation externe Karten-Apps wie Apple Maps, Google Maps oder Waze öffnen.

Dabei gelten die Datenschutzbestimmungen des jeweiligen Anbieters.

6. Backup und Export

Die App bietet Funktionen zum Export von Daten, zum Beispiel JSON oder CSV.

Dabei gilt:

  • Der Export erfolgt ausschließlich auf Initiative des Nutzers.
  • Die Verantwortung für die weitere Verwendung der exportierten Daten liegt beim Nutzer.
